Class SettingsManager

source code

This class provides an shallow interface for the rest of the program to use to interact with our configuration file.

Usage:

   Get settings:
       sm = SettingsManager()
       diff_tool = sm.get("external", "diff_tool")
       
   Set settings:
       sm = SettingsManager()
       sm.set("external", "diff_tool", "/usr/bin/meld")
